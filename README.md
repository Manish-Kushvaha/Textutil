# 📝 TextUtiles

TextUtiles is a simple and efficient web-based text utility tool that helps you analyze and manipulate text instantly. It is designed for students, writers, and professionals who need quick and easy text processing.

---

## 🚀 Features

- 🔤 Convert text to **UPPERCASE**
- 🔡 Convert text to **lowercase**
- ✂️ Trim extra spaces
- 📋 Copy text to clipboard
- 🧹 Remove extra spaces
- ❌ Clear text instantly
- 📊 Word and character counter
- ⏱️ Estimated reading time
- 🌙 Dark mode support

## 📖 Description

TextUtiles offers a simple and efficient way to analyze your text. Whether you want to count words, characters, or remove extra spaces, the tool provides instant results.

It is completely free with no hidden charges. No signup or login is required — just open the app, paste your text, and start using it.

The application works smoothly on all modern browsers like Chrome, Firefox, Edge, and Safari. It is responsive and lightweight, making it accessible on desktop, tablet, and mobile devices.
